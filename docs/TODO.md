# Recommended Docker Compose Applications

This document contains a curated list of Docker Compose applications that are highly useful for businesses, startups, and homelabs. They are categorized by use-case and are excellent candidates for future addition to the automated deployment suite.

## Core Infrastructure & Automation
- **Home Assistant**: Open-source home automation platform that puts local control and privacy first.
- **Node-RED**: Flow-based programming for the Internet of Things, wiring together hardware devices, APIs, and online services.
- **Nextcloud**: Self-hosted productivity platform for file synchronization, sharing, and collaboration.
- **Bind9**: Open-source DNS software system.
- **External-DNS**: Synchronize exposed Kubernetes Services and Ingresses with DNS providers (applicable to Swarm/Compose).
- **Ofelia**: A docker job scheduler (cron alternative).
- **Docker-Crontab**: A simple crontab for Docker containers.
- **Gotify**: A simple server for sending and receiving messages in real-time.
- **Chrony**: Versatile implementation of the Network Time Protocol (NTP).

## Monitoring & Observability
- **Uptime Kuma**: A self-hosted monitoring tool like "Uptime Robot".
- **cAdvisor**: Analyzes resource usage and performance characteristics of running containers.
- **Vigil**: Microservices Status Page & Monitoring system.
- **Gatus**: Automated developer-oriented status page.
- **Peekaping**: A simple status page.
- **InfluxDB**: Time series database for monitoring and analytics.
- **Graphite**: Enterprise-scale monitoring tool.
- **Zipkin**: Distributed tracing system.
- **SigNoz**: Open-source observability platform (APM, logs, traces).

## Media & Photos
- **Immich**: High performance self-hosted photo and video management solution.

## Search, Data & Analytics
- **Meilisearch**: Lightning-fast, open-source search engine.
- **Elasticsearch**: Distributed, RESTful search and analytics engine.

## Database Management
- **CloudBeaver**: Web-based cloud database GUI management tool supporting multiple databases.

## Identity & Access Management (IAM)
- **Authentik**: Open-source Identity Provider that emphasizes flexibility and versatility.
- **Keycloak**: Open-source identity and access management solution.
- **Tinyauth**: Lightweight authentication and authorization system.

## Endpoint Management & SIEM
- **Wazuh**: Open-source security platform for threat detection, endpoint security, and compliance.
- **FleetDM (Fleet)**: Open-source device management, built on osquery.

## File Transfer & Secure Sharing
- **SFTPGo**: Fully featured, highly configurable SFTP server with optional HTTP/S, FTP/S and WebDAV support.
- **Secure File Share (e.g., Pingvin Share / Firefox Send)**: Self-hosted file sharing platforms for secure, expiring attachments.
- **Secure Secret Share (e.g., Yopass / Hemera)**: Tools for sharing passwords and secrets securely via one-time encrypted links.

## AI Applications
- **Open Claw**: AI application platform.
- **Open WebUI**: Extensible, feature-rich interface for local LLMs (like Ollama).
- **LibreChat**: Free, open-source AI chat with multi-model support.
- **AnythingLLM**: Full-stack application to turn any document into an intelligent chatbot.

## Productivity & Knowledge Management
- **Wiki.js**: A modern, lightweight and powerful wiki app built on Node.js.
- **Pastebins (e.g., Hastebin / Microbin / Pastefy)**: Open-source text and code-snippet sharing.
- **RSS Reader (e.g., FreshRSS / Miniflux)**: Self-hosted RSS feed readers for staying up-to-date with news and blogs.

## Source Control & CI/CD
- **GitLab**: Web-based DevOps lifecycle tool that provides a Git-repository manager providing wiki, issue-tracking and CI/CD pipeline features.

## Security Scanning & Pen-Testing
- **Greenbone / OpenVAS**: Open-source vulnerability scanning and management system.
- **DefectDojo**: Open-source vulnerability management tool that streamlines the testing process.
- **OWASP ZAP (Zed Attack Proxy)**: Automated web application security scanner.
- **Kali Linux (via Kasm/VNC)**: Browser-based Kali Linux container for penetration testing environments.
- **Nessus**: Vulnerability assessment and scanning.
